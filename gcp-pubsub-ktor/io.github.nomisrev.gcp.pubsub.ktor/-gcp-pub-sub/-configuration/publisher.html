fun publisher(block: Publisher.Builder.(<Error class: unknown class>, <Error class: unknown class>) -> Unit)

Global Publisher.Builder configuration that will be applied to all Publisher's created by the GcpPubSub plugin. It's not possible to provide additional configuration on a per-publisher basis, since publishers are cached under the hood for performance reasons.

If you need to add configuration for a specific Publisher do so by checking projectId and topicId parameters of the lambda.
