fun topicAdmin(block: TopicAdminSettings.Builder.(<Error class: unknown class>) -> Unit)

Global TopicAdminSettings.Builder configuration that will be applied to all TopicAdminClient's created by the GcpPubSub plugin. It's not possible to provide additional configuration on a per-publisher basis, since publishers are cached under the hood for performance reasons.

If you need to add configuration for a specific TopicAdminClient do so by checking projectId parameters of the lambda.
